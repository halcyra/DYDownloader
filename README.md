# DYDownloader

[English](README.en.md)

DYDownloader 是一个抖音无水印原图/图集/动图/视频/账号/合集下载工具安卓 APP。

## 主要功能

- 抖音无水印原图、图集、动图、视频下载
- 作品、账号、合集链接解析与下载
- 本地解析 + 抖音官方接口，不依赖第三方解析平台或中转服务
- 基于 GPL-3.0 开源，许可证见 [LICENSE](LICENSE)

## Cookies 说明

部分内容解析与下载依赖 Cookies。

- **需要手动设置 Cookies：`设置 -> Cookies -> 登录抖音`**
- Cookies 数据仅保存在本地设备，应用仅在请求抖音时附加 Cookies，不会将 Cookies 上传到第三方平台或第三方服务
